# ATM_Reader
Console-based ATM Management System using Java and MySQL for ATM search and Cash Availability checking.

# ATM_Process_Simulation
ATM Reader Simulation is a Java-based project that simulates the real-world functioning of an Automated Teller Machine (ATM). It demonstrates essential banking operations such as user authentication, balance inquiry, cash withdrawal, cash deposit, and transaction processing.

The system validates account holders using ATM credentials and securely processes transactions while interacting with both the customer’s bank account data and the ATM’s internal database. Every transaction—such as withdrawals, deposits, and balance updates—is reflected in real time by updating the account holder’s bank details as well as the ATM database, including available cash records and transaction logs.

The project also simulates practical ATM scenarios like insufficient account balance, limited ATM cash availability, and transaction validation, providing a realistic representation of banking operations through Java and database integration.
